MODULE NAME
datatemplate

VERSIONS HISTORY
v1 created in July 17 based on previous module rawformat (in datamanager) used for extracting values according to a template. 

DESCRIPTION
This module extract and convert data from specified data sources (with a known data template) into tables 

TABLE OF CONTENT
crest: include functions to format data produced by crest (currently calibrations data)
coenergy: include functions to formata data provided by coenergy
